About This Repository
This repository contains all assignments, projects, and deliverables completed during the UofT Cybersecurity Boot Camp — a rigorous 24-week program covering the full spectrum of defensive and offensive security. Work was completed in a hypervisor-based lab environment and submitted via Git.

Highlighted Projects
🔵 ELK Stack Project (Week 13)
Deployed a cloud-based monitoring and SIEM solution on Microsoft Azure using the ELK stack. Configured Filebeat and Metricbeat to collect logs from multiple VMs, then created Kibana dashboards for threat visibility. View Project →
🔴🔵 Red vs. Blue Project (Week 20)
A full offensive/defensive simulation: attacked a vulnerable target machine using Metasploit and manual exploitation techniques, then switched roles to analyze the attack data in Kibana and propose defensive hardening measures. View Project →
